# 🏥 Radiology Patient Record System

This project is an API developed with **Node.js**, **Express**, **React**, **MySQL**, **HTML**, and **CSS**. Its goal is to optimize the **patient record registration process** for those undergoing X-ray exams in a hospital.

## 📋 System Description

The system allows radiology technicians to quickly and accurately register patient services. Through the **patient medical record ID**, personal data (name, date of birth, and gender) is automatically retrieved from an internal hospital server, making form completion easier.

In addition to recording patient personal data, the system also stores information such as the type of exam performed, the requesting department (e.g., Orthopedics, ICU, Internal Medicine, etc.), the technician responsible, date, time, and other relevant details.

## 🧭 Main Features

* 📄 **Registration form** with automatic data filling via medical record
* ✅ **Service registration** with exam and patient data
* 🕒 **List of the 20 most recent records** with edit and delete options
* 🔍 **Service history** with date-based filtering
* 📊 **Advanced reports** for supervisors, including dashboard cards and filters by:

  * Date
  * Patient origin
  * Patient gender
  * Exam type
  * And more

## 🛠️ Technologies Used

* **Frontend**: React, HTML, CSS
* **Backend**: Node.js, Express
* **Database**: MySQL
